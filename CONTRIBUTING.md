# Contributing

Thank you for considering a contribution to Literature Review Master. The project is intended to become stronger through use across disciplines, methods and research communities.

## Contribution principles

The skill should remain a literature intelligence system rather than a generic writing prompt. Contributions should improve decision quality, search quality, gap diagnosis, citation verification, journal positioning, or evidence discipline.

Please preserve the existing mode structure. Mode 0 to Mode E are the core interface and should not be renamed or replaced. New modes should be added only when they cover a genuinely distinct research situation.

## Useful contribution types

Good contributions include these.

- Field-specific search scope ladders.
- Better journal-family routing rules.
- More precise gap taxonomies.
- Examples from disciplines not yet well represented.
- Stronger templates for systematic, scoping or bibliometric reviews.
- Improved citation-verification statuses.
- Method-specific deep-reading checklists.
- No-data viability rules for theory, simulation, computation or conceptual work.
- Reviewer-attack examples based on real review patterns.
- Script improvements that make corpus logs, claim maps or journal filtering easier.

## What to avoid

Avoid contributions that make the skill more flattering but less accurate. Do not add language that encourages unsupported novelty claims. Do not add hard-coded journal recommendations that ignore fit. Do not call a review exhaustive without a protocol. Do not treat journal rank as a substitute for relevance.

## Suggested pull request format

Please include these items in a pull request.

```text
Purpose of the change:
Files changed:
Mode or agent affected:
Example use case:
Risk of unintended behaviour:
How the change was tested:
```

## Style

Use British English in documentation. Keep operational instructions direct. Avoid vague phrases such as "comprehensive review" unless the workflow actually supports comprehensive coverage.
