# Quick Start

Literature Review Master is a mode-driven skill for literature review, research-gap discovery, idea validation, related-work preparation and citation verification.

## Choose a mode

Use Mode A when you have a topic but no idea.

```text
Use Mode A. My broad topic is [topic]. Please map the literature, identify promising research gaps, flag killer-paper risks, and rank candidate ideas.
```

Use Mode B when you have a rough idea.

```text
Use Mode B. My idea is [idea]. Please run killer-paper analysis, framework-versus-cosmetic gap testing, and give a Pursue, Pivot, or Drop verdict.
```

Use Mode C when the project is chosen.

```text
Use Mode C. Please build a literature tree with direct predecessors, theoretical foundations, methodological foundations, adjacent streams and recent activity.
```

Use Mode D when you are writing.

```text
Use Mode D. Please prepare related-work material for [target journal family]. Include claim-citation mapping and British English prose.
```

Use Mode E before submission.

```text
Use Mode E. First extract all claims. Then verify citation existence, metadata, version risk, claim support, numerical claims, table claims, figure claims and internal evidence.
```

Use Mode J when the project has no real data.

```text
Use Mode J. I have no real data. Please decide whether this gap can be addressed with theory, modelling, proof, simulation or computation.
```

## Recommended output request

Ask for the shared research contract first.

```text
Before writing any final answer, please build the shared research contract with mode, domain, method family, evidence available, data constraint, target journal family, search scope, closest-collision risk and requested output.
```

## Verdict labels

For strategic decisions and idea validation, expect one of these.

```text
Pursue
Pivot
Drop-A, due to a conceptual failure
Drop-B, due to low strategic ROI
Drop-C, due to execution constraints
```

For citation verification, expect one of these.

```text
PASS
PASS WITH MINOR FIXES
MAJOR REPAIR REQUIRED
UNSAFE FOR SUBMISSION
```
