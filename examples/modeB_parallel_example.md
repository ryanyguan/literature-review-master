# Mode B Parallel Workflow Example

User request.

```text
I have a theory-driven idea about algorithmic pricing, strategic user response and numerical simulation. I have no real-world dataset. Is it worth doing?
```

Workflow.

```text
Literature Orchestrator builds the contract.
Search Query Architect creates direct and alternative queries.
Killer Paper Agent searches direct collisions.
Equilibrium Agent compares solution concepts and model structure if strategic interaction is central.
No-Data Viability Agent checks whether theory plus simulation can support the idea.
Journal Scope Agent routes the idea across the most plausible journal families.
Red Team Agent attacks the gap.
Integration Editor produces ROI and verdict.
```
