# Mode I. Systematic Survey or Review Paper

## Purpose

Use this mode when the user wants a systematic review, scoping review, bibliometric review, survey paper, or highly reproducible search protocol.

## Protocol elements

```text
Research questions
Databases and sources
Search strings
Time window
Inclusion criteria
Exclusion criteria
Screening stages
Coding scheme
Quality appraisal
Data extraction table
Synthesis method
Limitations
```

## Screening table

| Record | Source | Title | Year | Include or exclude | Reason | Notes |
|---|---|---|---|---|---|---|

## Coding scheme

Possible codes for the review topic include these.

```text
Domain
Method
Evidence type
Theory family
Construct family
Population or setting
Equilibrium concept if applicable
Optimisation or algorithmic method if applicable
Statistical or econometric design if applicable
Data availability
Simulation type
Target decision or policy object
Substantive implication
No-data viability when relevant
```

## Caution

Do not claim systematic coverage unless the protocol and screening records support it.
