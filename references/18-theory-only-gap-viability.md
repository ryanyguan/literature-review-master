# Mode J. Theory-Only Gap Viability

## Purpose

Decide whether a gap can become a publishable pure-theory, theory-plus-simulation, or algorithmic paper.

## Viability criteria

A theory-only gap is viable when these conditions hold.

```text
The missing mechanism is substantively important.
The mechanism changes a result, decision rule, equilibrium or method.
The closest literature does not already solve the same framework.
The model can produce a clean analytical object.
The paper can offer at least one strong proposition, theorem, equilibrium characterisation, comparative static, regime map, counterexample, or algorithmic result.
Numerical experiments can quantify the mechanism without pretending to be empirical validation.
Target journals accept theory or computational evidence for this kind of question.
```

## Rejection triggers

Recommend Drop or major Pivot if the gap is purely empirical, requires proprietary data, cannot change the theoretical prediction, or collides with a directly equivalent model.
