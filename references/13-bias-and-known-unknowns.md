# Bias and Known Unknowns

## Purpose

Literature review is vulnerable to anchoring, availability bias, field-name bias, confirmation bias and premature closure.

## Bias checks

Before finalising a verdict, ask these.

```text
Did we only search the user's preferred terminology?
Could another field use a different label for the same mechanism?
Did we over-weight recent papers because they were easy to find?
Did we ignore older foundational papers?
Did we confuse journal prestige with relevance?
Did we find papers that contradict the user's preferred framing?
Did we check working papers where the idea may already exist?
```

## Known unknowns

Every output must include a short section listing what remains unresolved.

```text
Unsearched databases
Unverified metadata
Potential adjacent fields
Papers requiring full-text reading
Claims that need manual confirmation
Recent working-paper risk
```

## No false closure

Do not write that there is no literature on a topic. Write that the current search did not identify a directly matching paper, and state the search boundaries.
