# Three-Pass Reading Protocol

## Pass 1. Triage

Use Pass 1 to decide whether a paper deserves deeper reading.

Extract these.

```text
Research question
Setting
Method
Main contribution claim
Closest relevance to the current project
Reason to deepen or discard
```

Pass 1 is enough for background papers and distant adjacent papers.

## Pass 2. Analytical reading

Use Pass 2 for core papers, close neighbours and possible killer papers.

Extract these.

```text
Precise model or empirical setup
Key assumptions
Identification, equilibrium or optimisation logic
Main results
Evidence type
Limitations
What the paper does not do
How the paper positions itself
```

For theory and game-theoretic papers, always record the solution concept and equilibrium object.

## Pass 3. Reconstructive reading

Use Pass 3 for papers that define the foundation, collide with the current idea, or provide a template worth emulating.

Tasks.

```text
Reconstruct the model or method.
Trace the proof or argument.
Identify which assumptions drive each result.
Map the paper's narrative structure.
Extract writing craft and contribution framing.
Identify reusable and dangerous citations.
```

## Reading-depth policy

Do not over-read low-relevance papers. Do not under-read close papers. The wrong reading depth creates false confidence.
