# Equilibrium Engine for Game-Theoretic Literature Review

## Purpose

For game-theoretic, mechanism-design, strategic-interaction, market, platform, policy, finance, operations, information-systems, engineering, public-sector and interdisciplinary models, the literature review must identify equilibrium structure rather than only topic similarity.

## Equilibrium extraction checklist

For each close paper, extract these.

```text
Players
Types
Actions
Timing
Information structure
Beliefs
Payoffs
State variables
Strategy space
Solution concept
Existence proof
Uniqueness or multiplicity
Equilibrium selection rule
Comparative statics
Welfare, profit, policy or performance analysis
Numerical equilibrium method
Boundary cases
```

## Common solution concepts

Use the correct concept for the model.

```text
Nash equilibrium
Bayesian Nash equilibrium
Subgame-perfect equilibrium
Perfect Bayesian equilibrium
Sequential equilibrium
Stackelberg equilibrium
Markov-perfect equilibrium
Wardrop equilibrium
Correlated equilibrium
Mixed-strategy equilibrium
Trembling-hand perfection
Mechanism-design incentive compatibility and participation constraints
Competitive equilibrium
Matching equilibrium
Learning equilibrium
Evolutionary equilibrium
```

Do not name a solution concept unless the literature and model justify it.

## Literature collision by equilibrium structure

Two papers may collide even if their application settings differ. Compare these.

```text
Best-response structure
Fixed-point equation
Threshold condition
Strategic substitutes or complements
Information asymmetry
Dynamic state update
Equilibrium regime partition
Welfare, policy or performance ordering
```

## Equilibrium gap types

Valid equilibrium gaps include these.

```text
A new information structure changes equilibrium behaviour.
A new timing structure changes subgame-perfect outcomes.
Heterogeneous agents create multiplicity or selection issues.
Operational, institutional or policy constraints alter strategic behaviour.
Demand-side or user-side behaviour changes market outcomes.
Financing, risk, contract or governance structures change incentive compatibility.
A numerical regime map reveals where analytical predictions reverse.
```
