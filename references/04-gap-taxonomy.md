# Gap Taxonomy

## Real gaps

A real research gap changes what the field knows. It is not merely a new context, new notation or new parameter.

Strong gap types include these.

```text
Mechanism gap, where a key force is absent from existing models.
Equilibrium gap, where strategic interaction changes the prediction.
Information gap, where beliefs, signals or observability change outcomes.
Constraint gap, where a real operational, financial or engineering constraint changes the feasible set.
Dynamic gap, where timing, learning or state evolution changes a static result.
Regime gap, where the literature assumes one regime and the important result lies elsewhere.
Method gap, where existing methods cannot solve a class of problems.
Evidence-structure gap, where the field has empirical regularities but no formal mechanism, or formal models but no numerical regime map.
Boundary gap, where a known result lacks conditions under which it fails.
Welfare or distributional gap, where profit and welfare predictions diverge.
```

## Weak gaps

Weak gaps include these.

```text
A new industry label with the same model.
A new dependent variable without a new mechanism.
A new parameter that does not change the solution structure.
A combination of literatures with no analytical tension.
A literature-count claim such as few studies have examined.
A gap that requires data the user does not have.
```

## Gap test

For every proposed gap, answer these.

```text
What exactly is missing?
Why does the missing element matter?
Which existing papers are closest?
Would adding the missing element change a theorem, equilibrium, algorithm, comparative static, welfare conclusion or managerial decision rule?
Can the gap be addressed without real data?
What would a hostile reviewer cite to collapse the gap?
```

If the answer does not change a model, result, method, theory or decision rule, the gap is likely cosmetic.
