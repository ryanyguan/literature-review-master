# Mode F. Rolling Update

## Purpose

Use this mode when a project has changed framing, paused for a period, or is approaching submission.

## Triggers

```text
The title changed.
The mechanism changed.
The target journal changed.
The application setting changed.
A new method was added.
Several months passed.
A reviewer asked for missing literature.
```

## Workflow

1. Compare old framing and new framing.
2. Generate new terminology and adjacent-field keywords.
3. Search recent papers and working papers.
4. Re-run killer-paper scan for the changed mechanism.
5. Update the literature tree and claim-citation map.
6. Mark old manuscript claims that must be weakened, removed or re-cited.

## Output

```text
New papers found
New risk level
Old claims requiring revision
New streams to cite
Verdict on whether the project remains viable
```
