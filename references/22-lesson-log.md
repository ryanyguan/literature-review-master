# Lesson Log Protocol

## Purpose

Maintain an append-only record of improvements, mistakes and project-specific terminology.

## Entries

Use this structure.

```text
Date:
Project:
Mode:
What failed:
What worked:
New search terms:
New dangerous papers:
New journal routing insight:
Rule to add:
```

## Rule

Never rewrite history. Append corrections. A literature review system improves by preserving failed searches and false starts.
