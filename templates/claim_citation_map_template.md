# Claim Citation Map Template

Use this map after Mode C, Mode D or Mode E. Its purpose is to connect manuscript claims with the exact evidence that supports them.

| Claim ID | Manuscript claim | Claim type | Location | Supporting source or internal evidence | Evidence location | Support strength | Risk | Action |
|---|---|---|---|---|---|---|---|---|

## Support strength labels

```text

```

```text
Wrong version

```
